# Picossci Branch 2B - 8ch 温度データロガー (RP2350)

スイッチサイエンスの「Picossci Branch 2B」基板（RP2350搭載）を使用し、最大8チャンネルの DS18B20（1-Wire温度センサー）からデータを取得して、SDカードにCSV形式でロギングするスケッチと専用ケースの3Dモデルデータです。

## 主な機能
- **8チャンネル独立/バス混在 1-Wire読み取り**: 独立した5つのGPIOと、3デバイスが並列接続された1つのGPIO（GP3）を統合管理。
- **SDカードへのCSVロギング**: 設定したインターバル（分単位）ごとに、全チャンネルの温度をタイムスタンプ付きで記録。
- **設定の保存機能**: インターバル時間は内部EEPROM(Flash)に保存され、再起動後も維持。
- **Core 1 による死活監視 (Watchdog)**: メイン処理(Core 0)が1-Wire通信等でフリーズした場合、別コア(Core 1)が検知して **赤色LEDを点灯** させます。


## 開発環境とボード設定
本スケッチは Arduino IDE でコンパイルします。RP2350のデュアルコア機能を使用するため、Earle F. Philhower 氏のコアを使用します。

1. **ボードマネージャURL**: `https://github.com/earlephilhower/arduino-pico/releases/download/global/package_rp2040_index.json`
2. **ボードパッケージ**: `Raspberry Pi Pico/RP2040/RP2350` をインストール。
3. **ボードの選択**: **`Pimoroni PGA2350`** を選択してください。（Picossci Branch 2Bに搭載されているマイコンと互換性があります）

### 必要なライブラリ
ライブラリマネージャなどで以下をインストールしてください。
- `LovyanGFX` (ディスプレイ描画用)
- `RTClib` (DS3231 RTC制御用)
- `OneWire` (1-Wireプロトコル通信用)
- `DallasTemperature` (DS18B20 温度変換用)
